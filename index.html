12-week hybrid program for strength + hypertrophy.

Weekly Structure

4-Day Split
Upper / Lower / Rest / Upper / Lower
3-Day Split
Upper / Lower / Full Hybrid

Day 1 – Upper Strength

Heavy Push / Pull
```
Bench Press
4x4–6
Weighted Pull-Ups
4x5–8
Overhead Press
3x5–8
Row (Barbell or Chest Supported)
3x6–8
Lateral Raises
3x12–15
Tricep Pushdowns
3x10–15
DB Curl (Optional)
2–3x10–12
```

Day 2 – Lower Strength

Squat + Posterior Chain
```
Back Squat
4x4–6
Romanian Deadlift
3x6–8
Leg Press / Split Squat
3x8–10
Leg Curl
3x10–15
Calf Raises
4x10–15
```

Day 3 – Hybrid Hypertrophy

Full Body Volume
```
Incline DB Press
3–4x8–12
Lat Pulldown
3–4x8–12
DB Shoulder Press
3x10–12
Cable Row
3x10–12
Lateral Raises
3x12–15
Face Pulls
3x12–15
Bicep Curl
3x10–15
Overhead Tricep Extension
3x10–15
Leg Press / Split Squat
3x10–12
Leg Extension / Curl
2–3x12–15
```

Day 4 – Lower Hypertrophy (Optional)

Lower Volume
```
Deadlift / Trap Bar
3x5–6
Bulgarian Split Squat
3x8–12
Leg Extension
3x12–15
Seated Leg Curl
3x12–15
Calf Raises
4x12–15
```

Progression

Weeks 1–4: Add weight or reps weekly
Weeks 5–8: Increase intensity (4x5 → 5x3)
Weeks 9–11: Heavy low reps
Week 12: Deload
