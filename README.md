# 💪 Hybrid Strength + Hypertrophy Program

A clean, mobile-friendly 12-week workout program designed to balance strength and muscle growth.

Built as a lightweight web app that can be saved to your phone and used in the gym.

---

## 🚀 Features

* Minimal, distraction-free design
* Optimized for mobile use
* Works on both desktop and phone
* Can be added to iPhone home screen (app-like experience)
* No login, no downloads, no apps required

---

## 📱 How to Use (iPhone)

1. Open the app link in Safari
2. Tap the Share button
3. Tap **"Add to Home Screen"**

Now it opens like a native app.

---

## 💻 How to Use (Desktop)

* Open the GitHub Pages link in any browser:

  * Chrome
  * Safari

---

## 🏋️ Program Structure

### Weekly Options

* **4-Day Split**

  * Upper / Lower / Rest / Upper / Lower

* **3-Day Split**

  * Upper / Lower / Full Hybrid

---

### Training Phases

* **Weeks 1–4:** Build volume and consistency
* **Weeks 5–8:** Increase intensity
* **Weeks 9–11:** Heavy strength focus
* **Week 12:** Deload
